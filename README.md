# Status View

用于治疗信息焦虑症的练手玩具。

## 功能

- 填入所需信息的接口，必须是JSON格式（目前）
- JSON定位语法，用于定位所需信息
- 字符串模板，用于自定义显示字段
- 配置备份和恢复功能
